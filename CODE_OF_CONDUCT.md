# Code of Conduct

## Our Standards

- Be respectful and constructive.
- Assume good intent and provide actionable feedback.
- Avoid harassment, discrimination, or personal attacks.

## Scope

This applies to project spaces including issues, pull requests, discussions, and community chat.

## Enforcement

Project maintainers may remove comments, lock threads, or ban contributors for behavior that violates this code of conduct.

## Reporting

Report unacceptable behavior privately to the maintainer with links and context.
